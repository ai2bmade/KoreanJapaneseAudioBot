# KoreanJapaneseAudioBot

Telegram listening-practice bot for Korean and Japanese expressions.

- Telegram: https://t.me/KoreanJapaneseAudioBot
- GitHub: https://github.com/ai2bmade/KoreanJapaneseAudioBot.git
- Free plan: 3 expressions per day
- Active plan: unlimited listening for manually activated users
